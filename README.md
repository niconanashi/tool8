# ニコ生CMメーカー 制作ツール

このページでは、画像を順番に表示する**スライドショー形式のCM**を作成し、ニコ生ゲームとして投稿できるファイル（`game.zip`）を簡単に作成できます。

### 使い方
* **画像の入れ替え・追加**：表示されている画像をクリックして差し替え、または「画像を追加」ボタンで追加します。
    * ※画像は **PNG / JPEG形式**、サイズは **1280x720 ピクセル固定** です。
* **順番の変更（重要）**：表示されている画像を**ドラッグ＆ドロップすることで、再生順を自由に入れ替えられます。**
* **BGMの設定**：音声ファイルを「BGMエリア」にドロップするだけで、ゲーム専用の形式へ自動変換して組み込みます。
* **詳細設定**：再生時間や、終了メッセージの内容・色を変更できます。
* **ダウンロード**：準備ができたら「game.zip を作成」ボタンでダウンロードし、ニコ生ゲーム投稿ページに投稿します。

### 仕様
* **画像サイズ**：1280x720 ピクセル（必須）
* **全体の再生時間**：**最大 200 秒まで**
    * 読み込み時間を含め、**190秒程度を上限**に設定することを推奨します。
    * ゲーム終了時間は200秒で固定です。（ `game.json` 内の `"totalTimeLimit": 200` の編集で短縮可能）
* **game.zip、game.json、 main.js の編集**
    * 右クリック→全てを展開で解凍しメモ帳などで編集、保存します。
    * filesフォルダとgame.jsonを同時選択した状態で右クリック→送る→圧縮（zip）により投稿するzipファイルになります。

---

### 注意事項
* **容量制限**：**投稿する game.zip には上限 30MB 程度の制限があります。音声、画像のファイルサイズにもよりますが 50～100 枚程度が上限の目安です。**
